---

### Step 2: 符号约定

- **张量**：15 或 -35 等具体的数
- **向量**：用粗体小写表示**a**, 若 **a** = \[2,3\]，则我们定义 $a^{(1)}=2$, $a^{(2)}=3$，同时在神经网络中，我们用 $x^{j}_{l,u}$ 表示第 $l$ 层神经元中第 $u$ 个单元的第 $j$ 维输出
- **矩阵**：用粗体大写表示 **A**

> 在矩阵乘法中，乘积的维度与矩阵维度相同

- $\underset{x \in S}{\arg \max } f(x)$：表示使 $f(x)$最大的数 $x$

---

### Step 3: 梯度计算

对$f(x) = a*x+b$，其中$x=[x^{(1)},x^{(2)}]$

那么其微分可以表示为：$\left[\frac{\partial f}{\partial x^{(1)}}, \frac{\partial f}{\partial x^{(2)}}\right]$
