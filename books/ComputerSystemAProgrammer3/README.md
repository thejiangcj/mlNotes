### 常用术语

- **位**：又称**比特**，一个由 0 和 1 组成的序列

- **字节**：由 8 个位组成的一组称为字节，常用于表示计算机中的文本字符——如英文字母等
- **文本文件**：全部由 ASCII 存储的文件
